Count++, a superior word count for atto.

QUICK INSTALL
=============

1. Install Dependency Plugin:

This plugin relies on statusbar plugin. Install that one first.
(See it's README.txt for installation documentation).

2. Install Count++

Put this entire directory at:

PATHTOMOODLE/lib/editors/atto/plugins/countplusplus

Visit your site notifications page to install the new plugins.

Add the "countplusplus" plugin to the Atto statusbar section.
