\assessment_form

The form to display the tab for sessions

Requires the argument 'sessions', which should be the array of sessions for the current course loaded from the database

For an example, see how it is instantiated in insview.php

To look at how deleting a recurring element is done, see definition_after_data and save_data. As well, see the elements was_deleted and deleteSession (the delete button) in add_session_repeat_template
